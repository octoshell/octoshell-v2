# Security Policy

## Supported Versions

We support version 2.8.x.

| Version | Supported          |
| ------- | ------------------ |
| 2.8.x   | :white_check_mark:                |
| 2.0.x   | :x: |

## Reporting a Vulnerability

  To report a vulnerability, please create an issue and specify:
  - Octoshell version,
  - Ruby version,
  - Postgresql version,
  - Redis version,
  - Sidekiq version,
  - How to reproduce a vulnerability:
    - preparations, if needed,
    - steps to reproduce,
    - impact.
